# Unified Schema

A unified schema that is part of HyAI data integration system comprising classes, object properties, and data properties.

# Contributors
The unified schema has been defined as an ongoing research effort by members of the TIB Leibniz Information Centre for Science and Technology, Leibniz​ University of Hannover, Centre for Individualised Infection Medicine (CIIM), TWINCORE-Centre for Experimental and Clinical Infection Research, and Hannover Medical School, Germany.​
